## Queue

```
1 <= 2 <= 3 <= add(element)

1 <= 2 <= 3 <= offer(element)

poll() <= 1 <= 2 <= 3
```

## Deque extends Queue
```
From Queue:
1 <= 2 <= 3 <= add(element)

1 <= 2 <= 3 <= offer(element)

poll() <= 1 <= 2 <= 3


Deque:

addFirst(element) => 1 <= 2 <= 3 

pollFirst() => 1 <= 2 <= 3 

1 <= 2 <= 3 <= addLast(element)

1 <= 2 <= 3 <= pollLast()
```
